XINS
XML Interface for Network Services

Description:

  • XINS is a web application framework.
  • XINS is design to be able to create web applications based on defined specifications.
  • XINS is based on a simple way to send requests (using URLs) and handle result (Simple XML format).
  • XINS generates a set of HTML pages from the specification and some test forms to test the application.
  • XINS generates Java code to invoke the web application and to develop it.
  • XINS also includes detects if the parameters are conform to the specification.
  • XINS includes concepts like load balancing, fail over, logging, security, properties and statistics.

Quick start:

  • Check that Java, ANT and Jetty are properly installed.
    Note that any other application server such as Tomcat or Orion could replace Jetty.
  • Set the XINS_HOME environment variable to the installed xins directory
  • Add %XINS_HOME%\bin to your PATH environment variable.
  • In the demo\xins-projects directory execute xins all-myproject.
  • Start Jetty in the same directory with with
    java -Dorg.xins.server.config=..\xins.properties -Djetty.home=%JETTY_HOME% -jar %JETTY_HOME%\start.jar ..\jetty_myproject.xml.
  • Run your browser to the web page demo/xins-project/specdocs/myproject/index.html
More detailed information can be found in the Documentation.
