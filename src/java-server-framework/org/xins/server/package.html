XINS server framework.

Logging

Logging is performed using Log4J. The following categories are logged to:
Category name Type of message
org.xins.server.LIFESPAN.BOOTSTRAP Bootstrapping of the XINS/Java Server Framework based on the build-time configuration settings.
org.xins.server.LIFESPAN.REINIT Initialization and re-initialization.
org.xins.server.LIFESPAN.SHUTDOWN Shutdown of the XINS/Java Server Framework.
org.xins.server.RUNTIME Runtime execution.
@since XINS 0.1
