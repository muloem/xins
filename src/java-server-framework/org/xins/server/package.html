XINS server framework.

Logging

Logging is performed using Log4J. The following categories are logged to:
Category name Type of message
org.xins.server.LIFESPAN.STARTUP Startup/initialization.
org.xins.server.LIFESPAN.REINIT Re-initialization.
org.xins.server.LIFESPAN.SHUTDOWN Shutdown/de-initialization.
org.xins.server.RUNTIME Runtime execution.
@since XINS 0.1
